# GitHub User Search App

This project is a simple **GitHub User Search** application built using **Vue 3** and the **Composition API**. It allows users to search for GitHub profiles by username and display relevant information, such as the user's bio, repository count, followers, and more.

## Features

- Search GitHub users by username.
- Display user details, including:
  - Name
  - Bio
  - Number of repositories
  - Followers and Following count
  - Location, Twitter handle, and website if available
- Simple and clean UI.

## Technologies Used

- **Vue 3** with Composition API
- **Fetch API** for data fetching
- **Tailwind CSS** for styling
